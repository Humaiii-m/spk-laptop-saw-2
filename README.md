# Sistem Pendukung Keputusan Pemilihan Laptop Menggunakan Metode SAW

Aplikasi web Sistem Pendukung Keputusan (SPK) untuk membantu menentukan laptop terbaik berdasarkan metode Simple Additive Weighting (SAW).

## Fitur

* Dashboard statistik
* Manajemen Kriteria
* Manajemen Alternatif Laptop
* Perhitungan SAW otomatis
* Matriks Keputusan
* Matriks Normalisasi
* Nilai Preferensi
* Ranking Laptop
* Export PDF
* Export Excel
* Dark Mode
* Responsive Design

## Teknologi

* HTML5
* Tailwind CSS
* Vanilla JavaScript
* LocalStorage
* Chart.js
* jsPDF
* SheetJS

## Metode

Metode yang digunakan adalah Simple Additive Weighting (SAW) dengan kriteria:

* Harga (Cost)
* RAM (Benefit)
* Processor (Benefit)
* Storage (Benefit)
* Bundling OS (Benefit)
* Jenis Laptop (Benefit)

## Cara Menjalankan

1. Clone repository
2. Buka menggunakan Visual Studio Code
3. Jalankan menggunakan Live Server
4. Akses melalui browser

## Deploy

Aplikasi dapat di-deploy menggunakan Netlify secara gratis.
